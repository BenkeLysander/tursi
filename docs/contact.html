Contact

For feedback, questions, and suggestions regarding Tursi please create a new issue in Tursi's GitHub Repo.

If you found a reproducible bug, spelling error or grammatical mistake, don't hesitate to inform me. I will try to fix it soon.
Please send me as much information as possible (operating system, Java runtime environment, error messages, files that you loaded and so on).

Suggestions for improvement are welcome, although I can not guarantee their implementation.
